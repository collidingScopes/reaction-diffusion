How to Use This Simulation

  • Click on the canvas to add patterns of chemical B
  • Use the dat.GUI controls to change simulation parameters in real-time:
    • Feed rate (F) and Kill rate (k) determine the pattern type
    • Diffusion rates control how quickly chemicals A and B spread
    • Time step affects ONLY simulation speed (not pattern formation)
    • Colors allow you to customize the visualization
  • Visualization modes:
    • Chemical A (red) - substrate that gets consumed
    • Chemical B (blue) - reactant that consumes A and replicates
    • Blend mode shows interaction between A and B
  • Try these patterns:
    • Coral-like patterns: F=0.055, k=0.062 (default)
    • Spots: F=0.03, k=0.062
    • Fingerprint-like stripes: F=0.030, k=0.059
    • Maze patterns: F=0.029, k=0.057
    • Bubbles: F=0.010, k=0.045
  • The simulation uses the Gray-Scott model, which mimics patterns found in nature
